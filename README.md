# Food Knowledge - more practice using APIs

## Made with:
Vue.js w/ Vue-CLI\
Bootstrap 4\
Axios\
AOS.js\
Edamam API - https://developer.edamam.com/food-database-api-docs

## What it does:
Connect API with axios to grab data through a search query\
Populate fields after query
